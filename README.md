# EOM-AI: Entropy-Oriented Mechanics for Battery Degradation Forecasting

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.XXXXXXX.svg)](https://doi.org/10.5281/zenodo.XXXXXXX)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Python 3.9+](https://img.shields.io/badge/python-3.9+-blue.svg)](https://www.python.org/downloads/)

**Entropy-Oriented Mechanics (EOM)** introduces the **entropic Lyapunov exponent** $\lambda_\text{EOM} = \dot{\Sigma}/\Sigma$ as a universal degradation metric. Using **NASA, Oxford, and CALCE datasets**, we reveal a **global stability attractor** at $(\lambda, \kappa, \Sigma) = (0,0,0)$ and an **entropy--impedance bridge** ($r = 0.71$).
